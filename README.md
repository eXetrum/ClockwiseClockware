# Getting Started with ClockwiseClockware App
#
### `git clone`
##

# To start server (backend)

### `cd server`
### `npm install`
### `node installDB.js`
### `npm run dev`
### `npm start`

# To start client (frontend)
### `cd client`
### `npm install`
### `npm start`

# Run both: server & client (root dir)
### `npm run dev`
